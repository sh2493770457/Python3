# Unauthorized Scan - Burp Suite 未授权访问检测插件

Unauthorized Scan 是一款针对 Burp Suite 的扩展插件，用于自动检测 Web 应用程序中的未授权访问漏洞。该插件通过移除认证头并比较响应差异，高效识别未授权访问漏洞，并提供丰富的报告和分析功能。

## 主要功能

- **智能检测**：通过删除认证头（如 Authorization、Cookie 等）并比较响应差异，准确发现未授权访问漏洞
- **多模式支持**：同时支持主动扫描和被动扫描模式
- **资源过滤**：智能过滤静态资源（图片、CSS、JS等），避免不必要的请求
- **丰富的报告**：提供详细的漏洞报告，支持导出为 Markdown 或 TXT 格式
- **可配置阈值**：自定义响应长度差异阈值，平衡检测的准确性和效率
- **友好的界面**：优化的UI布局，直观展示检测结果和请求/响应详情

## 快速开始

1. 下载 `Unauthorized_Scan.jar` 文件
2. 在 Burp Suite 中，进入 Extender -> Add 加载插件
3. 选择加载的 jar 文件，点击 Next 完成安装
4. 在 Burp Suite 顶部的选项卡中找到 "Unauthorized Scan" 标签

## 插件界面

插件主要包含以下界面组件：
- 配置面板：管理认证头列表、阈值设置和扫描选项
- 未授权漏洞表：显示检测到的所有未授权访问漏洞
- 所有流量表：展示所有捕获的请求，并支持筛选
- 请求/响应查看器：查看原始和未授权访问的请求与响应详情

## 最佳实践

1. 启用资源过滤功能，避免对静态资源进行不必要的检测
2. 调整响应长度差异阈值以适应不同的应用场景
3. 优先在测试环境中使用，避免在生产环境产生大量请求
4. 定期导出检测结果，方便后续修复和追踪

## 构建与开发

如需从源码构建：

```bash
mvn clean package
```

构建成功后，JAR 文件位于 `target/` 目录下。
