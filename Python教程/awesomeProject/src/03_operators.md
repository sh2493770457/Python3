# Go语言运算符

## 概述

本文档详细介绍了Go语言中的各种运算符，包括算术运算符、比较运算符、逻辑运算符、位运算符和赋值运算符等。运算符是用于在变量和值上执行操作的特殊符号，是编程语言中最基本的组成部分之一。

## 源代码

```go
// 03_operators.go - 运算符示例
package main

import "fmt"

func main() {
	// 算术运算符
	a, b := 10, 3
	
	fmt.Println("算术运算符示例:")
	fmt.Printf("a = %d, b = %d\n", a, b)
	fmt.Printf("a + b = %d\n", a+b)  // 加法
	fmt.Printf("a - b = %d\n", a-b)  // 减法
	fmt.Printf("a * b = %d\n", a*b)  // 乘法
	fmt.Printf("a / b = %d\n", a/b)  // 整数除法（结果为整数）
	fmt.Printf("a %% b = %d\n", a%b) // 取余
	
	// 浮点数除法
	c, d := 10.0, 3.0
	fmt.Printf("浮点数除法 %.1f / %.1f = %.2f\n", c, d, c/d)
	
	// 自增和自减
	x := 5
	x++ // 自增
	fmt.Printf("x自增后 = %d\n", x)
	x-- // 自减
	fmt.Printf("x自减后 = %d\n", x)
	
	// 比较运算符
	fmt.Println("\n比较运算符示例:")
	fmt.Printf("a == b: %t\n", a == b) // 等于
	fmt.Printf("a != b: %t\n", a != b) // 不等于
	fmt.Printf("a > b: %t\n", a > b)   // 大于
	fmt.Printf("a < b: %t\n", a < b)   // 小于
	fmt.Printf("a >= b: %t\n", a >= b) // 大于等于
	fmt.Printf("a <= b: %t\n", a <= b) // 小于等于
	
	// 逻辑运算符
	p, q := true, false
	
	fmt.Println("\n逻辑运算符示例:")
	fmt.Printf("p = %t, q = %t\n", p, q)
	fmt.Printf("p && q (与): %t\n", p && q)  // 逻辑与
	fmt.Printf("p || q (或): %t\n", p || q)  // 逻辑或
	fmt.Printf("!p (非): %t\n", !p)          // 逻辑非
	
	// 位运算符
	m, n := 60, 13 // 二进制: 60 = 00111100, 13 = 00001101
	
	fmt.Println("\n位运算符示例:")
	fmt.Printf("m = %d (%08b), n = %d (%08b)\n", m, m, n, n)
	fmt.Printf("m & n (按位与): %d (%08b)\n", m&n, m&n)    // 按位与
	fmt.Printf("m | n (按位或): %d (%08b)\n", m|n, m|n)    // 按位或
	fmt.Printf("m ^ n (按位异或): %d (%08b)\n", m^n, m^n)  // 按位异或
	fmt.Printf("m << 2 (左移): %d (%08b)\n", m<<2, m<<2)   // 左移
	fmt.Printf("m >> 2 (右移): %d (%08b)\n", m>>2, m>>2)   // 右移
	
	// 赋值运算符
	v := 10
	fmt.Println("\n赋值运算符示例:")
	fmt.Printf("初始 v = %d\n", v)
	
	v += 5  // 等同于 v = v + 5
	fmt.Printf("v += 5: %d\n", v)
	
	v -= 3  // 等同于 v = v - 3
	fmt.Printf("v -= 3: %d\n", v)
	
	v *= 2  // 等同于 v = v * 2
	fmt.Printf("v *= 2: %d\n", v)
	
	v /= 4  // 等同于 v = v / 4
	fmt.Printf("v /= 4: %d\n", v)
	
	v %= 2  // 等同于 v = v % 2
	fmt.Printf("v %%= 2: %d\n", v)
}
```

## 算术运算符

Go语言支持常见的算术运算符，用于执行数学计算：

| 运算符 | 描述 | 示例 |
|--------|-----|------|
| `+` | 加法 | `a + b` |
| `-` | 减法 | `a - b` |
| `*` | 乘法 | `a * b` |
| `/` | 除法 | `a / b` |
| `%` | 取余 | `a % b` |
| `++` | 自增 | `x++` |
| `--` | 自减 | `x--` |

### 注意事项：

1. 除法运算符 `/` 的结果取决于操作数的类型：
   - 如果操作数是整数，结果是整数（会舍弃小数部分）
   - 如果操作数是浮点数，结果是浮点数

2. 取余运算 `%` 只能用于整数类型

3. Go中的自增和自减是语句，不是表达式，不能在表达式中使用：
   - 正确：`x++`
   - 错误：`y = x++` 或 `++x`

## 比较运算符

比较运算符用于比较两个值，返回一个布尔结果（`true`或`false`）：

| 运算符 | 描述 | 示例 |
|--------|-----|------|
| `==` | 等于 | `a == b` |
| `!=` | 不等于 | `a != b` |
| `>` | 大于 | `a > b` |
| `<` | 小于 | `a < b` |
| `>=` | 大于等于 | `a >= b` |
| `<=` | 小于等于 | `a <= b` |

### 注意事项：

1. 不同类型的值不能直接比较，必须先进行类型转换
2. 结构体可以比较，但前提是每个字段都可比较
3. 数组可以比较，但切片、映射和函数不可比较

## 逻辑运算符

逻辑运算符用于组合布尔表达式：

| 运算符 | 描述 | 示例 |
|--------|-----|------|
| `&&` | 逻辑与 | `p && q` |
| `\|\|` | 逻辑或 | `p \|\| q` |
| `!` | 逻辑非 | `!p` |

### 短路求值：

Go支持短路求值：
- `&&`：如果第一个操作数为`false`，则不会评估第二个操作数
- `||`：如果第一个操作数为`true`，则不会评估第二个操作数

这在需要先检查条件防止错误的情况下非常有用，例如：
```go
if ptr != nil && ptr.Value > 10 {
    // 安全地访问ptr.Value
}
```

## 位运算符

位运算符对整数的二进制位进行操作：

| 运算符 | 描述 | 示例 |
|--------|-----|------|
| `&` | 按位与 | `m & n` |
| `\|` | 按位或 | `m \| n` |
| `^` | 按位异或 | `m ^ n` |
| `<<` | 左移 | `m << 2` |
| `>>` | 右移 | `m >> 2` |
| `&^` | 按位清除（位清除） | `m &^ n` |

### 位运算符的应用：

1. **按位与（&）**：只有当两个相应的二进制位都为1时，结果才为1
   - 常用于位掩码和检查特定位
   
2. **按位或（|）**：只要有一个相应的二进制位为1，结果就为1
   - 常用于设置标志位
   
3. **按位异或（^）**：两个相应的二进制位不同时，结果为1
   - 可用于简单的加密解密
   - 交换两个变量而不需要临时变量：
     ```go
     a = a ^ b
     b = a ^ b
     a = a ^ b
     ```
   
4. **左移（<<）**：将二进制位向左移动指定位数，右侧补0
   - 每左移一位，相当于乘以2
   
5. **右移（>>）**：将二进制位向右移动指定位数
   - 对无符号数，左侧补0
   - 对有符号数，左侧补符号位
   - 每右移一位，相当于除以2
   
6. **按位清除（&^）**：将第二个操作数中为1的位，对应第一个操作数中的位清零

## 赋值运算符

赋值运算符用于给变量赋值：

| 运算符 | 描述 | 示例 | 等价于 |
|--------|-----|------|--------|
| `=` | 简单赋值 | `v = 10` | |
| `+=` | 加法赋值 | `v += 5` | `v = v + 5` |
| `-=` | 减法赋值 | `v -= 3` | `v = v - 3` |
| `*=` | 乘法赋值 | `v *= 2` | `v = v * 2` |
| `/=` | 除法赋值 | `v /= 4` | `v = v / 4` |
| `%=` | 取余赋值 | `v %= 2` | `v = v % 2` |
| `<<=` | 左移赋值 | `v <<= 2` | `v = v << 2` |
| `>>=` | 右移赋值 | `v >>= 2` | `v = v >> 2` |
| `&=` | 按位与赋值 | `v &= 2` | `v = v & 2` |
| `\|=` | 按位或赋值 | `v \|= 2` | `v = v \| 2` |
| `^=` | 按位异或赋值 | `v ^= 2` | `v = v ^ 2` |

## 运算符优先级

Go语言中运算符的优先级从高到低如下：

| 优先级 | 运算符 |
|--------|--------|
| 5 | `*` `/` `%` `<<` `>>` `&` `&^` |
| 4 | `+` `-` `\|` `^` |
| 3 | `==` `!=` `<` `<=` `>` `>=` |
| 2 | `&&` |
| 1 | `\|\|` |

当有多个运算符时，同一优先级的运算符从左到右结合。如有疑问，可以使用括号 `()` 来明确表达式的计算顺序。

## 特别说明

1. Go没有三元运算符（`?:`），需使用if-else语句代替
2. Go中`++`和`--`是语句而非表达式，只能作为独立语句使用
3. 字符串可以用`+`连接，但不能使用其他算术运算符
4. Go不支持运算符重载

3. 使用逻辑运算符的短路特性编写条件语句
4. 使用不同的赋值运算符简化代码
