# Go语言入门 - Hello World

## 概述

本文档介绍了Go语言的第一个程序 "Hello World"，帮助您理解Go程序的基本结构和组成部分。

## 源代码

```go
// 01_hello_world.go - 第一个Go程序
package main

import "fmt"

// main函数是程序执行的入口点
func main() {
	// 使用fmt包中的Println函数打印到标准输出
	fmt.Println("你好，世界！") // 输出中文"你好，世界！"
	
	// 注释写法示例
	// 这是单行注释
	
	/* 
		这是多行注释
		可以跨越多行
	*/
}
```

## 代码详解

### 1. 包声明

```go
package main
```

每个Go程序都必须以`package`声明开始，它表明该文件属于哪个包。`main`包是一个特殊的包名，它告诉Go编译器这个程序可以被编译为独立可执行文件。一个可执行程序必须有一个`main`包。

### 2. 导入包

```go
import "fmt"
```

`import`语句用于导入其他包。这里导入了`fmt`包，它包含格式化输入输出的函数，类似于C语言中的`printf`和`scanf`。

### 3. main函数

```go
func main() {
    // 代码块
}
```

`main`函数是程序执行的入口点，类似于C、Java等语言。当程序启动时，会自动执行`main`函数中的代码。一个可执行程序必须包含一个`main`函数。

### 4. 打印语句

```go
fmt.Println("你好，世界！")
```

这行代码使用`fmt`包中的`Println`函数在控制台输出"你好，世界！"。`Println`会在输出后自动添加换行符。Go语言原生支持Unicode，所以可以直接打印中文字符。

### 5. 注释

Go支持两种注释方式：

1. 单行注释：以`//`开始，到行尾结束
   ```go
   // 这是单行注释
   ```

2. 多行注释：以`/*`开始，以`*/`结束，可以跨越多行
   ```go
   /* 
   这是多行注释
   可以跨越多行
   */
   ```

## 如何运行

在命令行中，进入源代码所在目录，执行以下命令：

```bash
go run 01_hello_world.go
```

你会看到输出：
```
你好，世界！
```

也可以先编译再执行：

```bash
go build 01_hello_world.go
./01_hello_world  # 在Windows上是 01_hello_world.exe
```

## 重点概念

1. Go是一种编译型语言，程序需要先编译成机器码才能运行
2. 每个Go程序都必须有一个`main`包和一个`main`函数
3. Go使用包（package）来组织代码
4. 函数使用`func`关键字定义
5. Go原生支持Unicode，可以直接使用中文等非ASCII字符
