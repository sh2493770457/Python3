### 生成ssh公钥

- win+r->cmd

![image-20250522095017610](../assets/image-20250522095017610.png)

- 挂载

```bash
fdisk -l
mkdir /test && mount /dev/sda1 /test
cd /test/root
mkdir .ssh
```

![image-20250522102810742](../assets/image-20250522102810742.png)

- 将生成的公钥粘贴到`.ssh`
